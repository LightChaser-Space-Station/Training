## 8. 图像处理入门 👁️

1. 安装 OpenCV：

```bash
pip install opencv-python
```

2. 编写程序加载图片并显示：

```python
import cv2

img = cv2.imread("example.jpg")
cv2.imshow("Image", img)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
